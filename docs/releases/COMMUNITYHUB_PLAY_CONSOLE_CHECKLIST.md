# CommunityHub Play Console checklist

## Technical

- [ ] Release AAB is signed with the upload key.
- [ ] `lintRelease` and `bundleRelease` pass.
- [ ] Play App Signing is enabled.
- [ ] Play signing SHA-256 is deployed in Digital Asset Links.
- [ ] Internal-test build opens fullscreen and verified links open CommunityHub.

## Policy and user data

- [ ] Public privacy policy is live.
- [ ] Data Safety form matches actual collection, sharing, encryption, and deletion behavior.
- [ ] In-app account deletion is functional.
- [ ] Public account-deletion URL is functional.
- [ ] App access instructions include a reviewer account if authentication blocks review.
- [ ] Ads declaration, content rating, target audience, and financial-features declarations are accurate.

## Store listing

- [ ] App name is `CommunityHub`.
- [ ] 512 × 512 Play icon uploaded.
- [ ] 1024 × 500 feature graphic uploaded.
- [ ] Phone screenshots uploaded.
- [ ] Category, contact email, short description, and full description completed.

## Testing and rollout

- [ ] Internal testing completed on multiple Android versions.
- [ ] Closed-testing requirement checked for this developer-account type and creation date.
- [ ] Crash/ANR pre-launch report reviewed.
- [ ] Production rollout starts staged rather than immediately at 100%.
