# Lifetopia CommunityHub Android

Android Studio Trusted Web Activity project for `https://community.lifetopiaworld.io`.

- Application ID: `io.lifetopiaworld.community`
- Android Browser Helper: `2.7.2`
- Android Gradle Plugin: `9.1.0`
- Gradle: `9.3.1`
- JDK: `17`
- Compile/target SDK: `36`

Open this directory directly in Android Studio. Read `docs/architecture/COMMUNITY_PHASE_10_1_ANDROID_STUDIO.md` from the monorepo root before signing or enabling the native wallet return scheme.
